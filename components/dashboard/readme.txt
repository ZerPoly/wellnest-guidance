components/dashboard

ConsultationStatistics.tsx - a visualization component that displays numerical data and trends regarding the total number of consultations attended within the selected timeframe.

ConsultationsTable.tsx - a comprehensive data table that lists individual appointment records. it supports tabs for "Upcoming" and "Previous" sessions, real-time search by student email, and multi-column sorting for dates and user IDs.

FlaggedStudents.tsx - a data-driven pie chart component that provides a high-level summary of student classifications. it visualizes the distribution of students across categories such as "In-Crisis," "Struggling," "Thriving," and "Excelling".

MoodSummaryStatistics.tsx - a dedicated reporting section that aggregates student-logged mood data to provide counselors with insight into the overall emotional climate of the student body.
